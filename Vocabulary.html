Vocabulary

Class

Classes are used to create different types of game objects, such as sprites, backgrounds, and game characters.
For example, you might create a class called Player that defines the properties and methods of a game character.
The Player class might have attributes such as the character's health, speed, and position, as well as methods for moving the character, attacking, and defending.

Iteration

Iteration refers to the process of looping over a set of instructions multiple times.
Iteration is a fundamental concept in programming, and it is used to perform repetitive tasks, such as processing large amounts of data and iterating over collections of objects.
In Python, there are several ways to perform iteration, including for loops, while loops.

Collision Detection

In pyarcade, collision detection refers to the process of detecting when two or more objects in a game or simulation come into contact with each other.
Collision detection is a fundamental concept in game development, and it is used to implement a wide range of game mechanics.
Such as collision-based scoring, damage calculation, and object interactions

Global vs Local Variables

A global variable is a variable that is defined outside of a function or class and can be accessed from anywhere in the program.
Global variables are useful for storing data that needs to be shared between different parts of a program A local variable, on the other hand, is a variable that is defined inside a function or class and can only be accessed from within that function or class.
Local variables are useful for storing data that is only needed temporarily and does not need to be shared with other parts of the program.

Exit

Stops the program

Physics Engine

A physics engine in pyarcade is to mimick the real life world.
It includes gravity, wall penetrability and many more.

Camera Movement

Camera movement refers to the screen moving with the sprite.
